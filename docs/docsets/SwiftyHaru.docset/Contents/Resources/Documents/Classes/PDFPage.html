PDFPage

public final class PDFPage

A handle that is used to manipulate an individual page.

Warning

If the PDFDocument object that owns the page is deallocated, accessing the page’s properties will cause a crash. The lifetime of a page should always be shorter than the lifetime of the document that owns the page.

  • draw(_:)

    Perform path drawing operations on the page.

    Warning

    The PDFPathContext argument should not be stored and used outside of the lifetime of the call to the closure.

    Precondition

    No drawing context must be present for this current page, i. e. you cannot run the following code:

    page.draw { context in

    page.draw { innerContext in
        // do things
    }
}
