Drawable

public protocol Drawable

Conforming types represent some high-level entities that can be put onto a page, like tables, grids etc. They must implement the draw(in:) method that does lower level things.

  • draw(in:position:)

    This method must perform drawing operations in the provided context. Use it to visualize your entity.

    Declaration

    Swift

    func draw(in context: DrawingContext, position: Point)

    Parameters

    context

    The context to perform drawing in.
