PDFPage

public final class PDFPage: _HaruBridgeable

A handle that is used to manipulate an individual page.

Warning

If the PDFDocument object that owns the page is deallocated, accessing the page’s properties will cause a crash. The lifetime of a page should always be shorter than the lifetime of the document that owns the page.
