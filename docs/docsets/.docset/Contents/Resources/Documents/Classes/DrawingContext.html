Path painting
  • clip(to:evenOddRule:_:)

    Sets the clipping area for drawing.

    Important

    Graphics parameters that are set inside the drawInsideClippingArea closure do not persist outside the call of that closure. I. e. if, for example, the context’s fill color had been black and then was set to red inside the drawInsideClippingArea closure, after the closure returns it is black again.

    Declaration

    Swift

    public func clip(to path: Path, evenOddRule: Bool = false,
                     _ drawInsideClippingArea: (Void) -> Void)

    Parameters

    path

    The path that constraints the clipping area. Must be closed.
    evenOddRule

    If true, uses even-odd rule for specifying a clipping area. Otherwise uses nonzero winding number rule.
    drawInsideClippingArea

    All that is drawn inside this closure is clipped to the provided path.
